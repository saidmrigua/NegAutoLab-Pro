# NegAutoLab Pro — UI Guide

## Overview

NegAutoLab Pro provides a desktop editing interface designed for film scanning, negative inversion, preview control, and export. The UI is organized to keep image browsing, previewing, and adjustment workflows clear and efficient.

---

## Main Interface Components

### Main Window
The main window is the central workspace of the application. It connects image browsing, preview rendering, adjustment controls, and export actions.

### Filmstrip Browser
The filmstrip browser displays loaded images as thumbnails and allows the user to switch quickly between documents in the current session.

### Preview Widget
The preview widget displays the currently selected image and supports interactive viewing features such as zoom, pan, crop overlays, and point-picking tools where available.

### Right Panel
The right panel contains the main adjustment controls used during editing. Depending on the current build and workflow, this may include:

- film mode controls
- white balance tools
- border balance / negative correction tools
- levels and tone controls
- LAB adjustments
- HSL controls
- crop tools
- export actions
- preset-related controls

### Histogram Widget
The histogram widget displays RGB histogram information for the active preview and helps the user evaluate tonal distribution while editing.

### Curve Widget
The curve widget provides tone curve visualization and, if enabled in the current build, curve editing support.

### Collapsible Sections
Controls are grouped into collapsible sections to keep the interface more organized and reduce visual clutter.

### Export Dialog
The export dialog is used to configure output settings such as file format, output location, profile options, and batch export behavior.

---

## Navigation

- Select images from the filmstrip browser to change the active document.
- Use the preview area to inspect the image visually while editing.
- Use the right panel sliders, combo boxes, buttons, and pickers to adjust the image.
- The preview updates as settings change, using the current preview rendering workflow.
- Use export controls when the image is ready for final output.

---

## Editing Workflow

1. Open one or more images
2. Choose the active image from the filmstrip
3. Set the appropriate film mode
4. Adjust negative correction and white balance
5. Refine tone, color, and LAB adjustments
6. Inspect the histogram and preview
7. Crop if needed
8. Export as TIFF or JPEG

---

## Interface Tips

- Use the filmstrip to move quickly between loaded images.
- Use presets to apply a saved look or starting point faster.
- Use the histogram as a visual guide during tonal adjustments.
- Use zoom and pan in the preview to inspect important image areas closely.
- All edits are non-destructive until export.

---

## Notes

- The preview is designed for responsive editing and may use proxy rendering.
- Final export uses the current image settings in the full-resolution output path.
- The exact control layout may evolve as the application continues development.
